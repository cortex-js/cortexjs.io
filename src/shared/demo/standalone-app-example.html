Example Standalone App

This demonstrates how the shared Header and Footer components work in a standalone React application.

Navigation Features:

  • Documentation Link: Takes users back to the main docs site
  • Compute Engine: Direct link to CE documentation
  • Tools Dropdown: Links to all interactive apps (editor, calculator, grapher, notebook)
  • GitHub: External link to the repository

Usage in React:

import { Header, Footer } from '../shared/components';

function App() {
  return (
    <>
      <Header currentApp="editor" standalone={true} />
      <main>
        {/* Your app content */}
      </main>
      <Footer />
    </>
  );
}
